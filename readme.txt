SHOPPING CART PROJECT

This project is a shopping website
that has products to sell and clients can
select the products and adding them into
the shopping cart then verify the payment
with credit cards or PayPal.
The project has authentication and
authorization in which clients are able to
register or login to the website and see
their orders they purchased.
Technologies that i used in the frontend:
HTML/CSS, javascript, bootstrap4,
Jquery,
Backend:
Node.js, MongoDB, PayPal api, Stripe api
